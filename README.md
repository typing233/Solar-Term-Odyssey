# 🌿 节气奥德赛 - 二十四节气文化探索之旅

一个结合2.5D交互与轻度RPG玩法的沉浸式Web应用，让青少年在探索奇幻场景与趣味解谜中，自然习得二十四节气与中国传统文化。

## ✨ 项目特色

### 🎮 游戏玩法
- **物候问答**：选择正确的物候现象与节气对应，了解二十四节气的自然规律
- **民俗匹配**：将民俗活动与对应的节气进行匹配，探索传统节日习俗
- **诗词填空**：点击选择正确的字词填入诗句空白处，感受古典诗词的魅力

### 🎨 视觉设计
- **2.5D手绘风格场景**：采用传统中国风配色和手绘边框效果
- **动态背景元素**：飘动的云朵、摇曳的树木、流动的河流
- **精美动画效果**：流畅的过渡动画、节日烟花庆祝效果

### 🏆 成就系统
- **三种难度等级**：简单、难度升级、超级难
- **勋章收集**：青铜探索者、白银守护者、黄金传承人
- **本地缓存**：用户进度和勋章信息通过浏览器localStorage存储

## 🚀 快速开始

### 环境要求
- Python 3.8+
- pip (Python包管理工具)

### 安装步骤

1. **克隆项目**
```bash
cd Solar-Term-Odyssey
```

2. **安装依赖**
```bash
pip install -r requirements.txt
```

3. **启动服务**
```bash
python app.py
```

4. **访问应用**
打开浏览器访问：`http://localhost:9832`

## 🎯 游戏说明

### 开始游戏
1. 点击"开始探索"按钮进入游戏
2. 在难度选择界面选择适合的挑战等级
3. 查看已获得的勋章和探索进度

### 游戏关卡

#### 第一关：物候问答
- 回答关于二十四节气物候现象的选择题
- 每题答对获得10分文化灵识
- 学习每个物候现象对应的节气知识

#### 第二关：民俗匹配
- 将左侧的节气与右侧的民俗活动进行匹配
- 每对匹配成功获得15分文化灵识
- 了解不同节气的传统习俗

#### 第三关：诗词填空
- 点击选择正确的字词填入诗词空白处
- 每空答对获得20分文化灵识
- 欣赏古典诗词的文学魅力

### 通关奖励
- 完成三关后获得对应难度的勋章
- 查看来自"时光长河"的过关密语
- 获得最终统计：文化灵识分数、正确答案数

### 数据存储
- 游戏进度和勋章信息存储在浏览器的localStorage中
- 清除浏览器缓存会丢失进度信息
- 支持多轮游戏挑战

## 🔧 API接口

### 获取游戏数据
```
GET /api/game-data?difficulty={easy|medium|hard}&level={1|2|3}
```

**参数说明：**
- `difficulty`: 难度等级（easy/medium/hard）
- `level`: 关卡编号（1/2/3）

**返回示例：**
```json
{
    "difficulty": "简单",
    "level": 1,
    "content": {
        "gameType": "quiz",
        "title": "物候问答",
        "description": "选择正确的物候现象与节气对应",
        "questions": [...]
    }
}
```

### 获取过关密语
```
GET /api/secret-message?difficulty={easy|medium|hard}&medal={勋章名称}
```

**返回示例：**
```json
{
    "message": "恭喜你在 2024年04月21日 完成了「简单」难度的挑战！...",
    "medal": "青铜探索者"
}
```

### 获取勋章列表
```
GET /api/medals
```
